﻿# ABCWarehouse

NopCommerce codebase

## Getting Started with Codespace

1. Create Codespace
2. Run `bash ./.devcontainer/init.sh`

## Running E2E Tests

End-to-end tests are located in the `e2e/` directory and use Playwright.

```bash
cd e2e
npm test
```

For more testing options, see the [e2e README](e2e/README.md).
